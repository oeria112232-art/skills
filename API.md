# API Documentation

## Authentication
- `POST /api/v1/auth/register` - Create new user
- `POST /api/v1/auth/login` - Authenticate and get JWT token

## Workshops
- `GET /api/v1/workshops` - List all workshops (Paginated)
- `POST /api/v1/workshops` - Create workshop (Admin/Instructor)

## Jobs
- `GET /api/v1/jobs` - List jobs (Paginated)
